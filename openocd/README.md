# The xPack OpenOCD

A standalone, cross-platform (Windows, macOS, GNU/Linux) binary
distribution of OpenOCD,
intended for reproducible builds.

## Project documentation

For information on how to install and use this project, please refer to the
[project website](https://xpack-dev-tools.github.io/openocd-xpack/).

## Project source

The source code of the current release is available on
[GitHub tag v0.12.0-7.1 tree](https://github.com/xpack-dev-tools/openocd-xpack/tree/v0.12.0-7.1).
